## Part 1:

For part 1, I created a 3D model that matches the isometric drawing included in with the exam package. I used Fusion 360, and I combined a mixture of additive and subtractive drawings. To create the model, additive drawings were used to extrude shapes to create the  form. Then, subtractive drawings were used to cut shapes out of the form. Examples of cuts include the 2 holes on opposing sides of the model, and the rail-like shape at the back of the model.

## Part 2:

For part 2, my student number is an even number. Therefore, I created a controller for Excitebike. The controller is meant to look like a motorcycle handlebar, keeping the theming consisitent with the game. The controller feautres one slide switch to allow the bike to move up and down between lanes, and A and B buttons. The A button allows the user to accelerate, and the B button allows the user to activate "turbo."
